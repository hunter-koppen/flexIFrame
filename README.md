## FlexIFrame
FlexIFrame is a widget that helps you embed an iframe to display external web content inside your app. Configure the URL and size, control security options (sandbox, referrerPolicy, allow), and use twoâ€‘way `window.postMessage` to send/receive data with the embedded page.

## Features
- Embed external content via a link
- Control security options
- Send messages through the iframe to the embedded page

## Usage
Add the widget to a page, set the URL, and ensure the target site allows embedding in iframes.
